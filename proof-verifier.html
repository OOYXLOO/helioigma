Run proof verifier

Check a Solstice Cipher completion proof.

Completed runs expose a run receipt in the form SC-4P-score-shifts-checksum. This page recomputes the checksum locally so judges can tell whether the receipt matches the published scoring formula. The default receipt is the stable one-click Demo Solve result.

Formula: solstice|4|score|shifts|4 is hashed with FNV-1a and converted into a six-character base-36 suffix.

This is a transparent run receipt for a public jam demo, not anti-cheat or identity proof. It exists so judges can replay the scoring path and inspect the checksum without trusting a screenshot.
