Theme Fit

The June solstice becomes the core timer: daylight is the player's resource, nightfall ends the run, and the final screen celebrates holding the longest day.

Turing Fit

The puzzle uses rotor-like alignment, binary and XOR glyph language, and target-versus-ring matching without pretending to simulate a historical machine.

Completed Loop

Four named phases, visible progress strip, score carry-over, streak scoring, shift counting, one-click stable Demo Solve proof, copyable and locally verifiable run proof, keyboard support, explicit node buttons, mobile layout, current WebM/GIF preview, and final run summary are ready.

Run Receipt

The stable judge path produces SC-4P-2907-62-Y5VFX1. The verifier recomputes the checksum and shows phases, score, shifts, and checksum facts.
